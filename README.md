# 💡 Project Overview

Sirene is a gamified web application focused on preserving, improving,
and expanding support for Philippine local languages through
community-powered speech and translation data collection.

The project is inspired by the theme **Tinig sa Liwanag**, which
highlights the importance of giving a voice to Filipino languages and
communities by creating digital resources that can support future
language technologies.

Many Philippine languages such as Cebuano, Ilocano, Hiligaynon, Waray,
Kapampangan, and others have limited digital resources. Because of this,
modern technologies such as speech recognition systems, translation
tools, and AI language models have difficulty understanding and
supporting these languages.

Sirene aims to address this challenge by creating an engaging platform
where users can contribute language data while improving their own
language abilities through missions, rewards, progression, and community
participation.

The platform combines two major concepts:

-   **Dataset Collection Platform** --- the main purpose of gathering
    high-quality speech, translation, and language data.
-   **Gamified Learning Platform** --- an engagement system that
    motivates users to continuously contribute.

------------------------------------------------------------------------

# 🚀 What We Will Develop

We will develop **Sirene**, a web-based gamified language contribution
platform where users can:

-   Translate English or Filipino phrases into Philippine languages.
-   Record their spoken translations.
-   Submit audio contributions for dataset creation.
-   Receive AI-based evaluation and feedback.
-   Earn experience points, coins, achievements, and mythical creature
    ranks.
-   Track their language learning progress.
-   Compete through leaderboards and community rankings.

The collected data will help create valuable resources for future
AI-powered language technologies.

------------------------------------------------------------------------

# 🎯 Main Goal

The main goal of Sirene is to create an accessible platform that
encourages communities to contribute high-quality language data while
promoting the preservation and advancement of Philippine languages
through an interactive and rewarding experience.

## 🤖 AI Evaluation System

After submission, the system evaluates:

-   Translation accuracy
-   Grammar
-   Naturalness
-   Pronunciation

Example Result:

Overall Score: 95/100

Rewards:

+95 XP\
+15 Coins

------------------------------------------------------------------------

## 📝 Transcript Verification

To maintain dataset quality:

1.  User records audio.
2.  Speech-to-text generates a transcript.
3.  User confirms or edits the transcript.
4.  AI evaluates the final verified version.

This ensures better-quality language datasets.
