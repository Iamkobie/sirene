# Sirene

> **Learning Languages. Building the Future of Philippine AI.**

Sirene is an AI-powered multilingual language learning platform designed to help users learn Philippine languages through interactive translation exercises, AI-powered speech evaluation, and gamified vocabulary activities.

Unlike traditional language learning applications, Sirene also serves as a multilingual dataset collection platform. Every completed learning activity generates an AI-annotated speech dataset that can contribute to future research on underrepresented Philippine languages.

---

# ✨ Features

- 🤖 AI-Generated Translation Exercises
- 🎤 AI Speech Evaluation
  - Fluency
  - Pronunciation
  - Completeness
  - Translation Accuracy
- 📚 Vocab Blitz (Daily Vocabulary Challenge)
- 🏆 Achievements & Progress Tracking
- 🌏 Multilingual Language Support
- 📊 AI-Annotated Multilingual Speech Dataset Collection

---

# 📖 Project Description

Sirene aims to address two major challenges:

1. **Providing an engaging platform for learning Philippine languages.**
2. **Building high-quality multilingual speech datasets for AI research.**

Each learning session produces contextual multilingual data, including:

- Source phrase
- Target language
- Spoken translation
- Audio recording
- Speech transcription
- AI evaluation scores
- Speaker metadata

This creates a bilingual, AI-annotated dataset that can support future developments in:

- Automatic Speech Recognition (ASR)
- Machine Translation
- Speech-to-Speech Translation
- Pronunciation Assessment
- Speaking Proficiency Evaluation
- Code-switching Research
